# Checkers Game States Reference

This document contains all possible states of the checkers game with preserved ANSI escape sequences for accurate color representation.

## Format

Each state section contains:
- **State Name**: Identifier for the state
- **Description**: What this state represents
- **Triggers**: How to reach this state
- **Key Visual Elements**: Important UI elements to verify
- **Raw Output**: Complete terminal output with ANSI escape sequences

## State Identification Guide

To identify current state programmatically:
1. Capture terminal output with colors preserved (`tmux capture-pane -p -e`)
2. Look for unique identifiers:
   - Welcome Screen: Contains "CHECKERS" ASCII art and "Press ENTER to begin"
   - Game Playing: Contains board grid with pieces "(w)" and "(b)"
   - Piece Selected: Contains green background highlights (RGB: 120,140,100)
   - Game Over: Contains "Game Over" popup box
   - With Hint: Contains "ğŸ’¡ Hint" box at bottom
